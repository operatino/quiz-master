# Quiz Master

https://operatino.github.io/quiz-master/

## Setup

```
git clone repo
cd quiz-master
npm i && npm run build
npm start
open localhost:8080
```

## Develop

```
npm run dev
```

## Build app

On UNIX system

```
./build-app.sh
```

## Deploy

Auto-deployed via Travis https://travis-ci.org/operatino/quiz-master to https://operatino.github.io/quiz-master using `deploy-gh.sh`.
